# Quick Start Guide

## Installation (5 minutes)

1. **Install dependencies:**
   ```bash
   npm run install-all
   ```

2. **Start the application:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   - Go to: http://localhost:3000
   - Login with:
     - Username: `admin`
     - Password: `admin123`

## Thermal Printer Setup

1. Complete a sale
2. Click "Print" on the receipt
3. Select your thermal printer
4. Set paper size to 80mm (3 inches)
5. Disable headers/footers in print settings

## Troubleshooting

**Can't login?**
- Default credentials: admin / admin123
- Check browser console for errors
- Verify server is running on port 5000

**Products not loading?**
- Check server logs
- Verify database is created (server/pos.db)
- Restart the server

**Images not showing?**
- Check server/uploads directory exists
- Verify file permissions
- Check browser console for 404 errors
